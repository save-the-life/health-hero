# 게임 메인 페이지 UI 구성

이 문서는 게임의 메인 페이지 UI 구성에 대해 설명합니다.

## 1. 전체적인 화면 구성

게임 메인 페이지는 크게 **상단 부분** (캐릭터 정보, 자원, 설정)과 **중앙 부분** (게임 진행 페이즈/스테이지 맵)으로 나뉩니다. 

배경은 밝고 활기찬 녹색 풍경에 노란색 돌길이 구불구불 이어져 있으며, 길을 따라 투명한 사각형 카드 형태로 게임 스테이지들이 배치되어 있습니다.

---

## 2. 상단 부분 (Header UI)

상단 부분은 모든 화면 상태에서 동일하게 유지됩니다.

### 2.1. 캐릭터 정보
- **위치**: 좌측 상단
- **표시 내용**:
  - 레벨별 캐릭터 이미지 (원형 아이콘)
  - 사용자의 현재 레벨 표시 (예: "Lv.20")
- **캐릭터 레벨별 외형**:
  - Lv.1: 의대 새내기
  - Lv.5: 인턴
  - Lv.10: 레지던트
  - Lv.15: 전문의
  - Lv.20: 헬스 히어로 (슈퍼히어로 의상)

### 2.2. 하트 (생명력)
- **위치**: 중앙
- **표시 내용**:
  - 빨간 하트 아이콘
  - 현재 보유한 하트 개수 (예: "5")
  - 다음 하트 충전까지 남은 시간 (예: "4분12초")
- **하트 시스템**:
  - 최대 보유량: 5개
  - 자동 충전: 5분마다 +1 (최대 5개까지)
  - 차감 조건:
    - 스테이지 진입 시: -1
    - 문제 오답 시: -1
  - 광고 충전: 광고 시청을 통해 하트 충전 가능
    - ⚠️ **추후 논의 필요**: 광고 충전 팝업 UI 및 표시 시점

### 2.3. 포인트
- **위치**: 하트 정보 우측
- **표시 내용**:
  - 노란색 별 아이콘
  - 현재 보유 포인트 (예: "10,200")
- **포인트 획득 방법**:
  - 문제 정답 시 난이도별 점수 획득
  - 스테이지 클리어 보너스
- **포인트 사용처**:
  - 아이템 구매 (오답 제거, 점수 2배, 자동 정답, 힌트)

### 2.4. 설정
- **위치**: 우측
- **표시 내용**:
  - 톱니바퀴 아이콘
- **설정 옵션**:
  - 소리 설정 (ON/OFF)
  - ⚠️ **추후 추가 예정**: 기타 설정 옵션

## 3. 중앙 부분 (게임 진행 맵)

중앙 부분은 게임의 진행 상황을 시각적으로 보여주는 맵입니다. 여러 개의 투명한 카드들이 구불구불한 길을 따라 배치되어 있으며, **각 카드는 하나의 페이즈를 나타냅니다**.

> ⚠️ **중요**: 스테이지 UI는 별도로 제공될 예정입니다. 메인 페이지의 카드는 페이즈 단위입니다.

### 3.1. 진행 방식
- **시작 위치**: 가장 아래쪽 페이즈 카드부터 시작
- **진행 방향**: 아래에서 위로 진행
- **잠금 해제 조건**: 
  - 첫 번째 페이즈의 모든 스테이지를 클리어해야 다음 페이즈가 잠금 해제됨
  - 순차적 진행 방식

### 3.2. 페이즈 카드 상태별 표시

#### 3.2.1. 활성/진행 중인 페이즈
- **시각적 특징**:
  - 투명한 카드 배경
  - 건물 아이콘이 명확하게 표시됨 (병원, 3층 건물, 2층 건물, 단층 주택 등)
  - 각 건물 아이콘 위에 빨간색 십자가 표시 (의료 테마)
  - "HOSPITAL"과 같은 텍스트 라벨 표시 (페이즈 카드의 일부, 단순 UI)
- **의미**: 현재 플레이 가능한 페이즈

#### 3.2.2. 잠금 상태의 페이즈
- **시각적 특징**:
  - 회색으로 흐릿하게 처리된 건물 아이콘
  - 주황색 물음표가 표시된 코인 모양의 아이콘으로 가려짐
  - 길이 점선으로 표시됨
- **의미**: 아직 잠금 해제되지 않은 페이즈

#### 3.2.3. 클리어된 페이즈
- **시각적 특징**:
  - 초록색 체크마크 표시
  - "완료" 텍스트 표시
  - 카드 내부의 건물 아이콘은 흐릿하게 처리됨
- **의미**: 페이즈 전체 클리어 완료

## 4. 명확화된 사항

### 4.1. 페이즈와 스테이지의 관계 ✅

**확정 사항**:
- **각 건물 카드는 페이즈를 나타냅니다**
- 스테이지 UI는 별도로 제공될 예정
- 페이즈 클릭 시 스테이지 선택 화면으로 이동

**구조**:
```
메인 페이지 (페이즈 카드들)
  └─ 페이즈 1 클릭
      └─ 스테이지 1 (5문제)
      └─ 스테이지 2 (5문제)
      └─ 스테이지 3 (5문제)
      └─ 스테이지 4 (5문제)
      └─ 스테이지 5 (5문제)
```

---

### 4.2. "완료" 표시의 적용 범위 ✅

**확정 사항**:
- **페이즈 전체가 완료되었을 때 페이즈 카드에 "완료" 표시**
- 페이즈 내의 모든 스테이지를 클리어해야 "완료" 표시

**시각적 표현**:
- 초록색 체크마크
- "완료" 텍스트
- 건물 아이콘 흐릿 처리

---

### 4.3. "HOSPITAL" 텍스트의 의미 ✅

**확정 사항**:
- **페이즈 카드의 일부로 단순 UI 요소**
- 별도로 신경쓰지 않아도 됨
- 페이즈별로 다른 텍스트 라벨 표시 가능

---

### 4.4. 추가 논의 필요 사항 (추후 제공 예정)

다음 항목들은 추후에 추가로 제공될 예정입니다:

- ⏳ 하트 충전 광고 UI 디자인 및 동작
- ⏳ 페이즈 클릭 시 동작 (스테이지 선택 화면 이동)
- ⏳ 스테이지 UI 디자인
- ⏳ 페이즈별 테마 구분
- ⏳ 진행 상황 표시 방법
- ⏳ 스크롤 및 네비게이션
- ⏳ 페이즈 잠금 해제 시 시각적 연출
- ⏳ 레벨업 시 캐릭터 변화 연출
